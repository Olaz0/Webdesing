    Project description

    "Laravel Store" is a website, similar to the OLX platform, that allows users to register, buy and sell items. The project is based on Laravel technology as the backend layer and HTML and CSS as the frontend layer. There is also an admin panel to manage users and sellers. The database was implemented in PHP.

    Screen shots

    Functions

    • User Registration: Visitors to the site can register as users. Registration may require basic information such as username, email address and password.
    • Buying Items: Registered users can browse the available items on the site and make purchases. They can search categories, view item details, place bids and complete transactions.
    • Sell items: Registered users can also list their items for sale. They can add descriptions, photos, set prices and accept offers from potential buyers.
    • Admin Panel: The admin has access to the admin panel where he can manage users, sellers, categories, offers and other page settings. The administrator can also moderate content and take action in case of violations of the rules.

    Technologies

    The following technologies were used in this project:

    • Backend: Laravel (PHP framework) - used to handle business logic, data management, user authorization, database and API interaction.
    • Frontend: HTML and CSS - responsible for the presentation layer, page layout, styling of user interface elements.
    • Database: PHP - used to store data on users, sellers, items, offers, etc.
