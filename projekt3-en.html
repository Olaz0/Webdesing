    • Project - PinPong

    Project description

    I wrote a PinPong game myself using the Processing tool. This is my interpretation of the classic retro Pong game that has been optimized for two players. In the game, players control paddles that move vertically, trying to bounce the ball and score points. The project was implemented using the Java programming language.

    Screen shots

    Functions

    • Two-player game mode: PinPong offers the opportunity to compete between two players. Each player controls his paddle, trying to keep the ball in play and scoring points by bouncing it off the opponent's paddle.
    • Mechanics of bouncing the ball: The ball moves around the board and bounces off the player's paddle and the side walls. Players must move their paddles appropriately to direct the ball towards the opponent and avoid losing a point.
    • Score Counter: The game keeps track of both players' scores and displays them on the screen. For each successful bounce of the ball by the opponent's paddle, the player receives a point. The player who scores a certain number of points wins the game.
    • Simple visuals: PinPong features retro-style graphics reminiscent of classic games from the 80s. Simple and clear visuals allow you to focus on the game.

    Technologies

    The following technologies were used in this project:

    • Processing: A tool for creating interactive applications and games. I used Processing to write a PinPong game and implement gameplay mechanics.
