    • Project - Escape From Ukraine

    Project description

    "Escape from Ukraine" is a game created in the Unity environment, using programming languages such as C ++ and CSS. This is an endless game inspired by the popular game "Subway Surfers". The player takes on the role of a character who runs through train tracks full of tanks and cars, trying to avoid obstacles and earn points.

    Screen shots

    Functions

    • Endless gameplay: The game offers continuous and dynamic gameplay, where the player's goal is to run as far as possible while avoiding obstacles.
    • Obstacles on the course: There are tanks and cars on the player's run route, which are obstacles. The player must jump, avoid collisions and move between vehicles to avoid failure.
    • Collecting Points: While running, the player has the opportunity to collect points, which can be earned in various ways, such as collecting bonus items or performing special actions.
    • Character Progression: The game may contain elements of character progression, such as unlocking new skills, clothes, or power-ups that help the player perform better.

    Technologies

    The following technologies were used in this project:

    • Unity: The environment in which the game is created, enabling the integration of various components and resources, creating the user interface, managing the game logic and graphics.
    • Programming languages: C++ - used to program game logic, event management and interaction with the player. CSS - used to style the user interface and graphic elements.
