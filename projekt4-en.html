    • Project - Restauracja C#

    Project description

    Our project is a program/code that emulates the cash register in a restaurant. The purpose of this solution is to enable the restaurant to simulate operations related to handling orders, generating receipts and storing order data in a database.

    Screen shots

    Functions

    • I've written a cash register emulator program to handle orders in a restaurant. The program includes a menu with different types of food and drinks that can be ordered. The menu includes a wide variety of dishes, such as starters, main dishes, desserts, drinks, etc. Each dish is described by name, description, price and other properties that may be relevant to the customer.
    • Placing an order: The program allows the user to place an order by selecting the appropriate items from the menu. The user has access to an interface where he can browse the available dishes and add them to the order. The system allows you to add many items to one order so that the customer can choose different dishes and drinks.
    • Receipt generation: After placing an order, the program generates a receipt that contains a description of the order, a list of ordered dishes and drinks, the total amount to be paid and the date of the order. The receipt can be displayed on screen or printed in paper form, providing the customer with order confirmation and payment information.
    • Receipt database: All placed orders are stored in the database. Each receipt is represented as a database entry containing information such as the receipt number, list of dishes ordered, date of order, etc. The database allows the restaurant to store and manage order data, which allows for later viewing of order history and data analysis.
    • Menu management: The program allows the restaurant owner to manage the menu. The owner can add, remove or edit existing menu items, such as dishes, drinks, prices, etc. This allows the menu to be adapted to changing customer preferences and updated depending on the needs of the restaurant.
